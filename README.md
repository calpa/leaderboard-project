# Leaderboard

![](https://i.imgur.com/GoFxoWk.png)

1. Install the packages using `yarn` in `root`, `website` and `server` folder.
2. Run `yarn start` in both `website` and `server` folder.

## Features
1. WebSocket
2. Pagination
3. Players with avatar images
4. Typescript
